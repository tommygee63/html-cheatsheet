HTML Elements

Tag Name Description
<p> Paragraph This tag is used to display text on a web page.
<h1>-<h6> Headers These tags are used to display headers with <h1> being the largest and <h6> being the smallest
<ul> Unordered List The <ul> tag is used to create a bullet pointed list using the <li> tag.
<ol> Ordered List The ordered list will create a numbered list.
<a> Anchor The anchor element is used to crate links to other parts of a web page, a web page within the web app or another web page.
<table> Table The <table> tag is used to create html tables
<div> Div The div element is used as a container to group parts of a web page
<img> Image The <img> tag is used to add an image onto a web page
<video> Video The video elemnt is used to include video content on a page, the controls attribute will display a basic control panel.
<strong> <em> Strong and Emphasis The <strong> element is used to create bold text. The <em> element will display text in italics.

Tables

Tag Name Description
<thead> Table Head The <thead> tag is used to define the rows in a table that make up the headings
<tbody> Table Body The table body contains the rows that make up the data within the table
<th> Table Header Table headers contain the title for each column in a table.
<tr> Table Row Table row container
<td> Table Data The container for the data in a table
